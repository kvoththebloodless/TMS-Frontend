The frontend module for Tell Me a Story which takes user stories and converts them into animations.

### `npm start`
This starts the node server and the website is available at localhost:3000.

### Enter the story lines

Enter the story line by line and watch it animate. Currently supported animations are "Shrug", "Pose", "bow" and "wave".
